What is this website?

This is an example website made using HTML and CSS, created to practice responsiveness – that is, the page works on both large and small screens. If you’re looking at this page on a laptop or desktop, you can test the effect by resizing your browser window.

The navigation bar above uses floats: at a width greater than 768 pixels (a so-called “typical breakpoint”), all the buttons fall to the left; below this width they stack on top of each other.

The cards below use flexboxes to achieve the same effect in a different way; again, below 768 pixels, the orientation of their container flips from “row” to “column” (side-by-side to on top of one another).

Github

The secondary function of this website is to practice using Github. Github is a widely-used tool for version control, allowing users to save different versions of their projects, and to restore progress from previously saved backups.

Github is also extremely useful for collaboration, allowing users to work on their own "branches" of a project, and subsequently "merge" their changes back into the original. While this project does not make use of this functionality, some future projects will.
